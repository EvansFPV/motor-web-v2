# License Choice

This project does not yet include a final license text.
Choose one of the options below before public release.

## Option 1: MIT
- Very permissive
- Minimal obligations (keep copyright + license notice)
- Good default for open-source web tools

## Option 2: Apache License 2.0
- Permissive + explicit patent grant
- Stronger legal clarity for contributors and commercial users
- Better choice if patent language matters

## Recommended Next Step
1. Pick MIT or Apache-2.0.
2. Add a `LICENSE` file with the full official text.
3. Update `README.md` License section.
